            GENERADOR DE CÓDIGOS DE CARNET - INSTALACIÓN
================================================================================

Bienvenido al instalador del Generador de Códigos de Carnet v1.0.0

Este asistente le guiará a través del proceso de instalación del software.

================================================================================
INFORMACIÓN IMPORTANTE ANTES DE INSTALAR
================================================================================

1. REQUISITOS DEL SISTEMA:
   - Sistema Operativo: Windows 10 o superior (64-bit)
   - Procesador: Intel Core i3 o equivalente
   - Memoria RAM: Mínimo 4GB (recomendado 8GB)
   - Espacio en disco: 500MB para la aplicación + espacio para datos
   - Resolución de pantalla: Mínimo 1024x768 (recomendado 1920x1080)

2. PRIVILEGIOS NECESARIOS:
   - Se requieren permisos de administrador para instalar el software
   - La aplicación puede ser ejecutada por usuarios sin privilegios administrativos

3. QUÉ SE INSTALARÁ:
   - Aplicación principal en: C:\Program Files\Generador de Códigos de Carnet
   - Carpeta de datos en: C:\ProgramData\Generador de Códigos de Carnet
   - Acceso directo en el escritorio (opcional)
   - Acceso directo en el menú Inicio
   - Desinstalador en el Panel de Control

4. ESTRUCTURA DE CARPETAS:
   
   Carpeta de Instalación (Program Files):
   - GeneradorCodigosCarnet.exe : Ejecutable principal
   - assets\                    : Recursos gráficos (logos, iconos)
   - config\                    : Archivos de configuración
   - DLLs y dependencias       : Bibliotecas necesarias
   - README.md                  : Documentación
   - LICENSE.txt               : Términos y condiciones
   
   Carpeta de Datos (ProgramData):
   - codigos_barras.db         : Base de datos principal
   - codigos_generados\        : Códigos de barras generados
   - carnets\                  : Carnets generados
   - backups\                  : Copias de seguridad
   - logs\                     : Archivos de registro
   - templates_carnet\         : Plantillas de carnets

5. CARACTERÍSTICAS DEL SOFTWARE:
   - Generación de códigos de barras (Code128, EAN13, EAN8, Code39)
   - Diseño y generación de carnets personalizados
   - Editor de plantillas HTML para carnets
   - Sistema de autenticación con roles (Admin, Editor, Visor)
   - Exportación a Excel
   - Gestión de empleados
   - Historial y auditoría
   - Copias de seguridad automáticas

6. PRIMER USO:
   - Al iniciar por primera vez, se le pedirá crear un usuario administrador
   - Guarde sus credenciales en un lugar seguro
   - Configure las opciones según sus necesidades

7. CONEXIÓN A INTERNET:
   - NO se requiere conexión a internet para el funcionamiento
   - Todos los datos se almacenan localmente
   - No se envía información a servidores externos

8. SEGURIDAD:
   - El software utiliza cifrado para las contraseñas
   - Los datos se almacenan localmente en su equipo
   - Se recomienda realizar copias de seguridad periódicas

9. ACTUALIZACIONES:
   - Las actualizaciones futuras se instalarán sobre esta versión
   - Los datos del usuario se preservarán durante las actualizaciones
   - Siempre se recomienda hacer backup antes de actualizar

10. DESINSTALACIÓN:
    - Use el desinstalador desde el Panel de Control
    - Los datos de usuario NO se eliminarán automáticamente
    - Si desea eliminar los datos, borre manualmente la carpeta de datos
